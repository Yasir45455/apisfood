Food API

API documentation for the Food App | Version 1.0.0

Base URL: https://foodeliverybackendnodejs.vercel.app

/products

GET Get all products

200: List of products

[
  {
    "_id": "product_id",
    "name": "Grilled Fish & Chips",
    "price": 18,
    "category": "Fish & Chips",
    "subCategory": "",
    "shortDescription": "Delicious grilled fish served with crispy chips"
  }
]
POST Add a new product

Request Body:

{
  "name": "Grilled Fish & Chips",
  "price": 18,
  "category": "Fish & Chips",
  "subCategory": "",
  "shortDescription": "Delicious grilled fish served with crispy chips"
}

201: Product created successfully

/products/{id}

PUT Edit product by ID

Path Parameter: id (string, required)

Request Body:

{
  "name": "Edited Fish & Chips",
  "price": 20,
  "category": "Fish & Chips",
  "subCategory": "",
  "shortDescription": "Delicious grilled fish with extra spices"
}

200: Product updated successfully

DELETE Delete product by ID

Path Parameter: id (string, required)

200: Product deleted successfully
